About PPI-Diff

PPI-Diff provides an online server for the co-design of protein-protein interaction (PPI) structures and sequences. Users can upload target protein features in .npz format to generate de novo peptide binders.

🧬 Feature Fusion: The server automatically extracts ESM-2 embeddings and performs resolution-aware feature fusion.
⚙️ Diffusion & Prediction: A conditional angle diffusion model generates the backbone structure, followed by an autoregressive sequence predictor.
📊 Output: The system returns a .pdb file (3D coordinates) and a .fasta file (amino acid sequence) for the generated binder.
